# Future Roadmap

This roadmap is directional and focused on realistic product/engineering expansion. Timelines are estimates, not commitments.

## Near-Term (0-3 Months)

- **Barcode-assisted meal logging**
  - Reduce manual entry friction and improve adherence.
- **Progress chart upgrades**
  - Better day/week/month trend clarity for calories, macros, hydration, and weight.
- **AI coach UX improvements**
  - Stronger response formatting, clearer action items, and better fallback states.
- **Widget refinements**
  - More useful glanceable summaries for daily goals.

## Medium-Term (3-6 Months)

- **HealthKit integration**
  - Optional sync of activity and body-related signals.
- **Workout recommendation layer**
  - Suggestion logic informed by consistency and recent performance patterns.
- **Premium subscription infrastructure**
  - Foundation for tiered features without fragmenting core experience.
- **Offline-friendly caching and sync**
  - Improved resilience in low-connectivity conditions.

## Stretch (6+ Months)

- **Deeper AI personalization controls**
  - User-tunable memory and coaching style preferences.
- **Advanced analytics package**
  - Rich correlation views across nutrition, training, and outcomes.
- **Adaptive goal planning**
  - Dynamic goal adjustments based on trend confidence and behavior consistency.
- **Platform expansion opportunities**
  - Potential broader ecosystem integrations if usage justifies investment.

## Engineering Enablers

Cross-cutting technical investments that support roadmap execution:

- service contract versioning,
- improved telemetry and diagnostics,
- stronger background processing patterns,
- and continued modularization to reduce feature coupling.
